# SkStudentPath - Kanpur's #1 Student Services Platform

Find verified coaching centers, PG accommodations, hostels, libraries & tiffin services in Kanpur.

## Getting Started

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`

## Features

- Browse verified student services in Kanpur
- Book coaching centers, PGs, hostels, libraries & tiffin
- List your services to reach thousands of students
- Secure authentication with OTP verification
